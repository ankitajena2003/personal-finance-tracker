# 💰 SaveSmart – Personal Finance Tracker

SaveSmart is a full-stack Personal Finance Tracker web application that helps users manage their **income, expenses, budgets, and balance** in one place with a clean, modern UI.

This project is built as a **resume-ready, interview-friendly project** using modern web technologies.

---

## 🚀 Features

### 🔐 Authentication
- User Registration
- User Login with JWT Authentication
- Protected Routes (Unauthorized users redirected to Login)

### 📊 Dashboard
- Total Income
- Total Expenses
- Remaining Balance
- Welcome popup for new & returning users

### 💸 Expense Management
- Add expenses
- View expense list
- Delete expenses
- Category & date based entries

### 💰 Income Management
- Add income sources
- View income list
- Delete income entries

### 🎯 Budget Management
- Set monthly budgets by category
- Track spent vs limit
- Budget overview

### 👤 My Account
- View profile details
- Edit name & email
- Add personal information
- Sidebar profile sync (name updates everywhere)

### 🎨 UI & UX
- Fully responsive layout
- Modern dark theme
- Smooth animations
- Sidebar navigation
- Mobile-friendly design
